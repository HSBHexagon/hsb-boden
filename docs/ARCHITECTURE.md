# Repository Architecture

## Tech Stack
*   **Framework:** Astro
*   **UI Library:** React
*   **Styling:** Tailwind CSS
*   **Testing:** Vitest
*   **Deployment:** Cloudflare Workers

## Directory Structure
*   `src/components/`: Reusable UI components, split by category (forms, layout, sections).
*   `src/data/`: Data models and content files defining site structure and information.
*   `src/layouts/`: Base layout wrapper for pages.
*   `src/lib/`: Utility functions, tracking, and validation logic.
*   `src/pages/`: File-based routing for the Astro application.
