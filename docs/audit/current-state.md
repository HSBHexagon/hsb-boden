# HSB Ist-Audit

Stand: 2026-06-03

## Live-Seite

- `https://hsb-boden.de/` ist WordPress/Elementor.
- Yoast SEO und Sitemap sind aktiv.
- Sitemap enthält aktuell im Wesentlichen die Startseite.
- Die aktuelle Seite ist indexierbar, aber stark als Onepager strukturiert.

## Wichtigste Lücken

- Zu wenige eigenständige SEO-Landingpages.
- Überschriftenstruktur muss sauberer werden.
- Bilder benötigen beschreibende Alt-Texte.
- Referenzen sollten als freigegebene Karte/Fallstudien ausgebaut werden.
- Formular sollte projektqualifizierende Felder enthalten.
- Google Unternehmensprofil, GA4 und Search Console müssen eingerichtet werden.
