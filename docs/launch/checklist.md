# Launch-Checkliste

## Vor Staging

- `npm run test:run`
- `npm run check`
- `npm run build`
- `npm run deploy:dry-run`
- Desktop/Mobile prüfen
- Referenzfreigaben prüfen

## Vor Livegang

- WordPress-URL-Inventar exportieren
- Redirects festlegen
- Cloudflare DNS vorbereiten
- Google Search Console Property prüfen
- GA4/GTM verbinden
- Google Unternehmensprofil aktualisieren
- Datenschutz/Cookie-Texte prüfen
- Formularziel produktiv verbinden

## Nach Livegang

- HTTP 200 für Kernseiten prüfen
- Sitemap abrufen
- Robots prüfen
- Formular-Testanfrage senden
- GA4-Events prüfen
- Search Console Indexierung anstoßen
